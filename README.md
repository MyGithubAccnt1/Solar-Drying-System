## Setup

Open a terminal - `Shortcut key:` Ctrl + `

```bash
  cd .\FRONTEND\`
```

Install dependecies

```bash
  npm i
```

Run the project

```bash
  npm run dev
```

Open a new terminal - `Shortcut key:` Ctrl + Shift + `

```bash
  cd .\BACKEND\
```

Install dependecies

```bash
  npm i
```

Run the project

```bash
  npm run start
```
